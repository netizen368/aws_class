 HTML 문서는 태그로 구성
 태그는 다음과 같은 형태로 되어있음
 - 단일 태그 : <태그명 속성="값" 속성="값">
 - 쌍 태그 : <태그명 속성="값" 속성="값"></태그명>
 - 속성은 모든 태그가 가질 수 있는 공통 속성과 태그마다 종류가 다른 속성이 있음
 - 속성의 값은 속성마다 정해져 있는 경우도 있고, 정해져 있지 않은 경우도 있음
 - 값은 앞뒤로 '' 또는 ""을 붙임. 생략도 가능한데 이 경우 값에 공백이 있으면 안됨
 - VS code 에서는 태그를 입력할 때 태그명 입력 후 엔터를 치면 쉽게 태그를 입력가능

 html 태그
 - html 문서 나타내는 태그
 - 속성
  - lang : 문서의 언어를 설정. en : 영어, ko : 한국어
 
 head 태그
 - 문서와 관련된 정보를 모아놓은 태그
 - 키워드, 작성자, 타이틀, 파비콘, 화면 너비 설정, css,js 연결 등
 
 meta 태그
 - 문서 정보를 나타내는 태그
 - charset="UTF-8" : 문자 인코딩 방식을 UTF-8로 설정
 - name="viewport" content="width=device-width, initial-scale=1.0"
 - 화면 지정 방식을 설정
 - 화면 너비를 장치 너비의 1배를 기준으로 잡음
 - 모바일에서 글자가 작아지는걸 방지할 수 있음
 
 title 태그
 - 문서의 타이틀을 설정
 - 브라우저 좌측 상단의 문구를 지정
 
 body 태그
 - 화면을 구성하는 태그 
 - 구성하고 싶은 내용을 body 태그 안에 배치
a태그
 - 페이지를 이동할 때 사용하는 태그
 - 기본 값
  - 글자색 : 방문 전 파랑, 방문 후 - 보라, 클릭 중 - 빨강
  - 밑줄 : 있음
 - 속성
  - href : 이동할 위치를 지정
   - URL 주소 입력(http나 https 포함) : 다른 사이트로 이동
   - URL 주소 입력 : 같은 사이트 다른 페이지로 이동
   - tel:전화번호 : 전화번호 연결 - 모바일에서
   - mailto:이메일 주소 : 이메일 작성으로 연결
   - #아이디명 : 같은 페이지에서 다른 위치로 이동 
  - target : 링크 페이지가 표시될 위치
   - _self : 기본값, 현재창
   - _blank : 새창이나 새탭
   - _parent : 부모 프레임
   - _top : 최상위 조상 프레임
  - download : 파일을 다운로드할 때 사용
   - 값을 지정하지 않으면 서버에 있는 파일명을 그대로 다운로드
   - 값에 파일명을 지정하면 지정된 파일명으로 다운로드
  - rel : 현재 문서와 링크된 문서의 관계를 나타내는 속성
   - noopener : 링크된 문서가 현재 문서를 참조하지 못하게 함 - 보안 강화
   - noreferrer : 링크된 문서에 출처를 전달하지 않음 - 보안 강화
   - nofollow : 검색 엔진이 링크를 따라가지 않도록 지시 : SEO(검색 엔진 최적화)
   - external : 외부 링크
   - author : 저자
   - help : 도움말
   - license : 라이센스 정보
  경로 지정 방식
 - 절대 경로
  - 최상위 폴더를 기준으로 위치를 잡음
  - 최상위 폴더가 변경되면 기존 경로를 변경해야할 수 있음
 - 상대 경로
  - 현재를 기준으로 위치를 잡음
  - 최상위 폴더가 변경되도 기존 경로가 바뀌지 않음
  - ../ : 상위 폴더
시멘틱 태그
 - 기능 없음
 - 의미를 부여할 때 사용
 - 종류
  - header 태그 : 상단 콘텐츠, 웹 문서의 머리말
  - nav 태그 : 메뉴를 그룹화
  - figure 태그 : 멀티미디어 콘텐츠를 그룹화
  - main 태그 : 중요한 영역, 문서의 핵심 영역
  - aside 태그 : 부가 콘텐츠 영역
  - section 태그 : 콘텐츠를 그룹화
  - article 태그 : 개별 콘텐츠
  - footer 태그 : 하단 콘텐츠
서식태그
 태그 종류
 - block 태그 
  - 새라인에서 시작, 해당 라인 전체를 공간으로 잡음
  - 너비/높이를 이용하여 크기 변경 O
  - 시멘틱 태그, h1~h6, p, div 등
 - inline 태그 
  - 현재 위치에서 시작
  - 너비/높이를 이용하여 크기 변경 X
  - 글자 크기를 이용하여 높이는 변경하는 것처럼 보일 수 있음
  - a, span, i,b 등
 - inline-block 태그 
  - 현재 위치에서 시작 
  - 너비/높이를 이용하여 크기 변경 O 
  - img, input 등
 h1~h6 태그
  - 제목 태그
  - 숫자가 작을 수록 큰 글씨
 p태그
 - 문단을 지정하는 태그
 - 기본값
  - 위 아래 margin이라는 여백이 있음
  - p태그끼리 위아래로 배치된 경우 여백이 하나로 통합

 hr 태그

 img 태그
 - 이미지를 넣을 때 사용
 - inline-block 태그
 - 속성
  - src : 가져올 이미지 주소
  - alt : 이미지 로딩 실패 시 보여줄 문구이자 스크린 리더기가 읽을 문구
  - width : 이미지 너비. 기본값은 원본 이미지 너비의 100%, auto 지정 가능
  - height : 이미지 높이. 기본값은 원본 이미지 높이의 100%, auto 지정 가능
 - width/height의 auto 사용법
  - width나 height 중 하나를 고정으로 하고, 나머지를 auto로 설정해서 
    이미지 비율이 깨지지 않게 함
