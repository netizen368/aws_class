<스타일 적용 방법>
 방법1. link 태그를 이용
 미리 만들어 놓은 스타일을 가져와서 사용(css 파일)
 최초 1회만 다운로드, 이후에는 캐시에서 가져다 사용
 
 방법2. style 태그를 이용
 같은 문서에서 style 태그를 이용하여 사용
 문서가 로딩될 때 스타일이 함께 로딩
 
 방법3. style 속성을 이용
 모든 태그에 style 속성을 사용할 수 있음
 우선순위가 가장 높음. 
 비추천
 스타일 적용하기
 선택자{
 속성명 : 값;
 속성명 : 값;
 }

<기본 선택자>
 선택자
 요소를 선택하는 방법
 
 스타일 적용
 선택자{
 속성명 : 값;
 속성명 : 값;
 }
 
 1. 전체 선택자
 *
 모든 요소를 선택
 우선 순위 : 0점
 
 2. 태그 선택자
 태그명 
 태그들을 선택
 우선 순위 : 1점
 
 3. 클래스 선택자
 .클래스명
 class 속성의 값으로 클래스명이 들어가는 요소를 선택
 class 속성은 모든 태그에서 사용 가능한 속성으로 특별한 기능은 없음. 스타일 적용할 때 사용
 우선 순위 : 10점
 
 4. 아이디 선택자
 #아이디명
 id 속성 값이 일치하는 요소를 선택
 우선 순위 : 100점

<자식자손 선택자>
 자식 선택자 
 부모선택자 > 자식선택자
 요소 안에 있는 자식을 선택
 
 자손 선택자 
 조상선택자 자손선택자
 요소 안에 있는 자손을 선택
 자식 요소도 포함
 
 우선 순위가 같으면 밑에 있는 속성이 적용

<동위 선택자>
 동위 선택자
 같은 부모를 가진 요소를 선택
 형제 요소를 선택
 
 선택자A + 선택자B
 선택자A 바로 뒤에 있는 선택자B를 선택
 
 선택자A ~ 선택자B
 선택자A 뒤에 있는 선택자B를 선택
 
 한 요소를 구체적으로 선택하는 선택자
 선택자를 공백없이 나열
 태그명이 들어가는 경우 태그명을 제일 앞에 작성

<속성 선택자>
 속성 선택자
 요소의 속성을 이용한 선택자
 [속성명] : 해당 속성을 가진 요소를 선택
 [속성명=값] : 속성의 값이 일치하는 요소를 선택
 
 문자열 속성 선택자
 속성에 특정 문자열이 있는지 확인할 때 사용
 
 [속성~=A]
 속성의 값이 A를 단어로 포함하는 요소를 선택
 class="A B" O
 class="A-B" X
 class="B-A" X
 class="B A" O
 class="AB"  X
 class="BA"  X
 
 [속성|=A]
 속성의 값이 A-단어로 시작된 요소
 class="A B" X
 class="A-B" O
 class="B-A" X
 class="B A" X
 class="AB"  X
 class="BA"  X
 
 [속성^=A]
 속성의 값이 A로 시작하는 요소를 선택
 class="A B" O
 class="A-B" O
 class="B-A" X
 class="B A" X
 class="AB"  O
 class="BA"  X
 
 [속성$=A]
 속성의 값이 A로 끝나는 요소를 선택
 class="A B" X
 class="A-B" X
 class="B-A" O
 class="B A" O
 class="AB"  X
 class="BA"  O
 
 [속성*=A]
 속성의 값이 A를 포함하는 요소를 선택
 class="A B" O
 class="A-B" O
 class="B-A" O
 class="B A" O
 class="AB"  O
 class="BA"  O

<구조 선택자>
 형태 구조 선택자
 몇번째 위치에 있는 요소인지를 선택
 태그명과 함께 사용. 클래스나 아이디들 다른 선택자는 의미가 없음
 of-type이 포함

 태그명:nth-of-type(숫자)
 태그들 중 숫자번째 위치에 있는 요소

 태그명:nth-of-type(수열)
 태그들 중 수열번째 위치에 있는 요소
 예 : 2n(짝수번째 요소를 선택), 2n-1(홀수번째 요소를 선택)

 태그명:first-of-type
 태그들 중 첫번째 위치에 있는 요소
 nth-of-type(1)과 같은 결과

 태그명:last-of-type
 태그들 중 마지막 위치에 있는 요소
 
 일반 구조 선택자
 태그 종류 상관없이 순서에 맞는 요소를 선택
 child를 포함
 nth-child(숫자), nth-child(수열), first-child, last-child

<반응 상태 선택자>
 반응 선택자
 선택자:hover
 요소에 마우스를 올렸을 때 
 선택자:active
 요소를 클릭했을 때
 
 상태 선택자
 선택자:checked
 checked된 상태인 요소를 선택
 input태그 중 checkbox나 radio가 체크된 상태
 
 선택자:focus
 요소가 활성화된 상태
 
 선택자:enabled
 요소가 사용가능한 상태
 
 선택자:disabled
 요소가 사용 불가능한 상태
  
<문자 선택자>
 선택자::selection
 드래그된 글자들을 선택
 
 선택자::before
 요소 앞에 글자를 추가
 글자 이외의 다양한 작업을 할 수 있음
 content 속성이 필수. 앞에 추가할 문자열
 
 선택자::after
 요소 뒤에 글자를 추가
 글자 이외의 다양한 작업을 할 수 있음
 content 속성이 필수. 뒤에 추가할 문자열
 float 속성을 제어할 때 사용
