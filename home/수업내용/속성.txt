<스타일 단위>
 - px : 고정 픽셀
 - % : 부모 요소의 %. 100%가 부모 크기
 - em : 부모 요소의 배수. 1em이 부모 크기
 - rem : 최상위 요소의 배수(html태그)의 배수
 - vh : view height, 현재 브라우저의 높이를 100으로 나눈 값이 1vh
 - vw : view width, 현재 브라우저의 너비를 100으로 나눈 값이 1vw
 - vmin/vmax : vh와 vw 중 최소값/최대값
 - deg : 각도

<폰트 속성>
 color
 - 글자 색상
 
 색상 지정 방식
 1. 미리 지정된 색상 이름
 - red, blue, green 등과 같이 100개가 넘는 이름
 
 2. 16진수로 표현
 - #rrggbb 형태로 표시
 - rr은 빨강 색상, gg는 녹색 색상, bb는 파랑 색상 정보를 16진수로 표현(0~9,a~f)
 - 각 rr값이 같고, gg값이 같고, bb값이 같으면 #rgb로 줄여 쓸 수 있음
 - red => #ff0000 => #f00
 - white => #ffffff => #fff
 - black => #000000 => #000
 
 3. rgb함수/rgba함수
 - rgb(숫자,숫자,숫자), rgba(숫자,숫자,숫자,투명도)
 - 숫자는 0~255
 - 투명도는 0~1사이의 실수. 1이면 불투명, 0이면 투명
 - #f00 => rgb(255, 0, 0) => rgb(255, 0, 0, 1)
 
 4. hsl(숫자,숫자,숫자), hsla(숫자,숫자,숫자,투명도)
 - 색상, 채도, 명도를 이용하여 색상을 표시 
 
 font-weight
 - 폰트 굵기
 - 100~900사이의 값을 100단위로 입력
 - normal : 400
 - bold : 700
 
 font-size
 - 폰트 크기
 - 숫자와 단위를 함께 사용
 
 font-family
 - 폰트 종류를 지정
 - 일반적으로 여러 폰트를 한번에 지정
 - 왼쪽 폰트가 있으면 적용, 없으면 오른쪽 폰트로 이동
 - 보통 제일 오른쪽은 기본 폰트 중 하나를 선택(monospace, serif, sans-serif)
 - 폰트 이름이 한글인 경우 영문명도 함께 작성
 - 폰트 이름이 여러 단어인 경우 ''나 ""를 반드시 붙여야 함
 
<텍스트 속성>
 text-decofation
 - 글자에 줄 긋기
 - 값
 - underline : 밑줄
 - overline : 윗줄
 - line-through : 취소선(가운데줄)
 - none : 없음
 
 white-space
 - 공백 처리 방법을 설정
 - 값
 - normal : 여러 공백을 하나로 처리. 기본값
 - nowrap : 여러 공백을 하나로 처리. 줄 바꿈 없이 한줄 표시
 - pre : 여러 공백을 그대로 처리
 - pre-line : 여러 공백을 하나로 처리. 엔터 구분. 영역 넘어가면 자동 줄 바꿈
 - pre-wrap : 여러 공백을 그대로 처리. 영역 넘어가면 자동 줄 바꿈
 
 letter-spacing 
 - 글자 간격
 
 word-spacing
 - 단어 간격
 
 line-height 
 - 한 줄 높이. 한 줄에 글자를 상하 기준으로 가운데 배치
 
 text-align 
 - 글자 정렬 방법
 - 값
 - center, left, right, justify(양쪽 정렬)
 
 text-overflow
 - 영역을 벗어난 텍스트를 표시할 방법을 설정
 - 값
 - clip : 자름
 - ellipsis: 말줄임(...)
 
 overflow
 - 영역을 벗어난 요소를 보여줄지 말지를 결정
 - 값
 - visible : 보여줌. 기본값
 - hidden : 안보이게 해줌
 - scroll : 스크롤을 이용
 - auto : 요소가 넘어가면 스크롤이 추가
 
<글자 정렬>
 text-align
 - inline, inline-block 요소를 정렬하는 방법(글자 포함)
 - 값
 - left(기본값), center, right, justify(양쪽 정렬)
 
<영역 속성>
 box-sizing
 - width/height의 기준을 정하는 속성
 - 값
 - border-box : 너비/높이가 컨텐츠 + 패딩 + 테두리를 합친 width/height
 - content-box : 기본갑, 너비/높이가 컨텐츠의 width/height
 
 width/height
 - 요소의 너비/높이를 지정
 - box-sizing 값에 따라 의미가 달라짐
 
 margin
 - 요소의 테두리 밖 여백을 지정하는 속성
 - margin-top, margin-right, margin-bottom, margin-left를 줄여 쓴 단축 속성
 - 방향은 위부터 시계방향. 상 > 우 > 하 > 좌
 - 액자의 위치를 설정
 - 사용 예시
 - margin : 10px 20px 30px 40px
 - 상 : 10px, 우 : 20px, 하 : 30px, 좌 : 40px
 - margin : 10px 20px 30px => 좌우가 대칭
 - 상 : 10px, 우 : 20px, 하 : 30px, 좌 : 20px
 - margin : 10px 20px => 상하 동일, 좌우 동일
 - 상 : 10px, 우 : 20px, 하 : 10px, 좌 : 20px
 - margin : 10px => 상하좌우 동일
 - 상 : 10px, 우 : 10px, 하 : 10px, 좌 : 10px
 - margin : 0 auto
 - 위아래 0, 좌우 자동 배치 => width가 고정px일 때 가운데 배치
 
 padding
 - 요소의 테두리 안 여백을 지정하는 속성
 - 액자안의 사진 시작 위치를 지정
 - 사용 방식은 margin과 동인
 - padding-top, padding-right, padding-bottom, padding-left를 줄여쓴 단축 속성
 
 테두리
 위치 : top, right, bottom, left
 border-width
 - 테두리 두께
 - border-위치-width를 줄여쓴 단축 속성
 - 순서는 위를 기준으로 시계방향
 
 border-color
 - 테두리 색상
 - border-위치-color를 줄여쓴 단축 속성
 - 순서는 위를 기준으로 시계방향
 
 border-style
 - 테두리 종류
 - border-위치-style를 줄여쓴 단축 속성
 - 순서는 위를 기준으로 시계방향
 - 값
 - solid, dashed, dotted, none, hidden, double, groove, ridge, inset, outset 등
 - solid(실선), none(없음)을 많이 사용
 
 border
 - 모든 테두리 정보를 한번에 지정하는 속성
 - border-width, border-color, border-style을 줄여쓴 속성
 - 모든 테두리를 동일하게 지정
 - 사용예시
 - border : 1px solid black;
 
 border-위치
 - 각 위치의 테두리 정보를 한번에 지정하는 속성
 - 사용예시
 - border-위치 : 1px solid black;
 
 border-radius
 - 둥근 테두리를 지정하는 속성
 - border-위치1-위치2-radius를 줄여 쓴 단축 속성
 - 위치1 : top, bottom, 위치2 : left, right
 - 왼쪽위 모서리를 기준으로 시계방향
 - 보통 단위로 px을 사용, %를 이용하면 원이나 타원을 만들 수 있음
 
<배경 속성>
 background-color
 - 배경색
 
 background-image
 - 배경 이미지
 - 값
 - url(이미지경로)
 
 background-repeat
 - 배경 이미지 반복
 - 보통 요소가 배경 이미지보다 큰 경우가 의미가 있음
 - 값
 - repeat : x측, y측 반복
 - repeat-x : x측 반복
 - repeat-y : y측 반복
 - no-repeat : 반복 안함
 
 background-position
 - 배경 이미지의 시작 위치를 지정
 - x측 방향 y측 방향순으로 위치를 입력
 - 값
 - 고정px, %
 - left, center, right : x측 방향
 - top, center, bottom : y측 방향 
 - 사용 예시
 - background-position : 100px 100px
 - background-position : center
 
 background-size
 - 배경 이미지 크기를 변경
 - 너비 높이 순으로 입력
 - 값
 - 고정px, %
 - auto : 이미지 원본 크기
 - contain : 이미지가 요소 안으로 들어갈 수 있게 확대/축소, 여백이 생길 수 있음
 - cover : 요소를 이미지가 덮을 수 있도록 확대/축소. 여백이 안 생김
 
 background-attachment
 - 배경 이미지가 움직이지 않게 고정하는 속성
 - 값
 - scroll : 기본값, 배경 이미지가 움직임
 - fixed : 배경 이미지가 움직이지 않게 고정
 
 background-clip
 - 박스 모델에서 어디까지 배경으로 볼지를 결정하는 속성
 - 값
 - border-box : 테두리까지 배경으로 봄
 - padding-box : 패딩까지 배경으로 봄
 - content-box : 컨텐츠까지만 배경으로 봄
 
 background-origin
 - 배경 이미지를 배치할 기준
 - 값
 - border-box : 테두리부터 배경 이미지 배치
 - padding-box : 패딩부터 배경 이미지 배치
 - content-box : 컨텐츠만 배경 이미지 배치
 
 background
 - color image repeat attachment position clip origine size를 한번에 적용하는 단축 속성

<float 속성>
 float
 - 요소를 왼쪽, 오른쪽으로 배치할 때 사용하는 속성
 - inline, block에 상관 없이 왼쪽, 오른쪽으로 배치
 - 엔터에 의한 공백을 없애줌
 - 요소에 width를 지정하지 않으면 요소 안에 있는 내용만큼 너비가 잡힘 
 - 부모 요소의 높이가 잡히지 않음
 
 clear
 - float 속성이 적용된 요소가 있으면 다음 요소에 영향을 미치는데 이를 초기화 하는 속성
 - 값
 - left : 왼쪽 float 요소의 높이를 계산한 후 배치
 - right : 오른쪽 float 요소의 높이를 계산한 후 배치
 - both : 왼쪽과 오른쪽 float 요소의 높이 중 긴 요소를 기준으로 배치
 
 calc() 함수
 - 서로 다른 단위를 계산할 때 사용
 - 연산자 앞 뒤로 공백을 반드시 써야 함
 - +, -, *,
 
<display 속성>
 display
 요소의 display를 변경하는 속성
 값
 inline
 inline-block
 block
 none : 안보이게 감춤
 각 태그 마다 기본 display값이 있는데 이를 변경하는 속성
 
<position 속성>
 position
 요소를 배치하는 방식을 지정하는 속성
 값
 static : 기본값
 fixed
 요소를 브라우저를 기준으로 위치를 고정
 위로가기 버튼
 top, bottom, left, right 속성을 이용하여 위치를 조절
 + 값이면 요소 안쪽으로 향하고, - 값이면 요소 바깥쪽으로 향함
 relative
 요소의 상대위치를 계산해서 배치
 요소가 원래 있어야할 위치에서 top, bottom, left, right 속성값에 따라 이동
 자손 요소 중 position:absolute인 요소의 기준점이 된다
 absolute
 기준 요소를 기준으로 배치
 top, bottom, left, right 속성을 이용하여 위치를 조절
 기준 요소
 absolute가 적용된 요소의 조상 요소 중 
 position:relative나 position:absolute인 요소가 있는 가장 가까운 조상이 기준요소
 기준 요소가 없으면 브라우저가 기준
 top, bottom, left, right를 이용하여 width와 height를 대신할 수 있음
 absolute가 적용된 요소는 부모 요소 높이에 영향을 주지 않음
 
<z-index 속성>
 z-index
 요소들이 서로 겹쳤을 때 누구를 위로 보낼지 결정하는 속성
 숫자가 클수록 위로 올라감
 숫자가 같으면 html코드 순서상 뒤에 있는게 위로 올라감(형제 요소들인 경우)
 요소들은 부모 요소를 넘을 수 있음
 
<opacity 속성>
 opacity
 요소의 투명도를 설정
 0~1사이의 실수
 0이면 투명, 1이면 불투명
 부모 요소에 투명도가 적용되면 자식 요소에도 적용됨
 자식 요소에 투명도를 1로 해도 투명이 됨
 배경 색을 rgba로 이용하는게 좋을 수도 있음
